Attributs "id", "class" et "name"

Attribut "id"

Permet de donner un identifiant à un élément HTML

Util pour cibler un élémnet HTML avec du JavaScript

Permet d'identifier une ancre

Permet de lier une balise "label" à un champ de formulaire, pour amélioré l'expérience utilisateur.



Attribut "class"

Permet d'identifier des élément HTML par un nom de classe.

Peut être multiple dans le document.

Plusieurs classe peuvent etre appliquées à un élément.
