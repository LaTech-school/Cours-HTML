Les formmulaires

Déclaration d'un formulaire

On déclare un formulaire avec les balises <form></form>

Les attributs de la balise <form>

action
e.g. : <form action="quelquepart.php"></form>
Défini l'adresse du fichier qui traitera les données du formulaire.
method
e.g. : <form method="GET"></form>
e.g. : <form method="POST"></form>
Défini la méthode de requete HTTP pour transmettre les données au serveur.
GET : Les données seront transmises au serveur via l'URI http://site.com?firstname=Bobby
POST: Les données seront transmises au serveur via l'entete de la requete http://site.com
novalidate
e.g. : <form novalidate></form>
Indique au navigateur de ne pas s'occupper du controle du formulaire.
